# SysSet

**Declarative System Setup Tools as Dev Container Features and Standalone Installers**

[![Static Badge](https://img.shields.io/badge/Quickstart-darkgreen?style=for-the-badge)](https://quantized8.github.io/sysset/quickstart)
[![Static Badge](https://img.shields.io/badge/Features-darkgreen?style=for-the-badge)](https://quantized8.github.io/sysset/features)
[![Static Badge](https://img.shields.io/badge/Contribute-darkgreen?style=for-the-badge)](https://quantized8.github.io/sysset/contribute)

</div>

<hr>

<p align="justify">
<strong>SysSet</strong> is a collection of software installers and system setup tools – collectively <strong>features</strong>.
Each feature is designed to be compatible with a wide range of environments,
including macOS and various Linux distributions, in containers and on bare-metal machines.
Features are distributed as both <a href=https://containers.dev/implementors/features/><strong>devcontainer features</strong></a> and
<strong>self-contained/bundled installers</strong>
published to <a href=https://docs.github.com/en/packages/working-with-a-github-packages-registry/working-with-the-container-registry><strong>GitHub Container Registry (GHCR)</strong></a> and <a href=https://docs.github.com/en/repositories/releasing-projects-on-github/about-releases><strong>GitHub Releases</strong></a>, respectively.
They provide a seamless experience for installing and configuring essential software,
with rich configuration options that cater to a wide range of use cases and requirements.
Features are thoroughly tested and documented, with a consistent design and user experience across the board.
</p>
