# Background

Conceptual references that explain environment and shell behavior across Linux, macOS, devcontainers, CI, and automation contexts.

- {doc}`env-vars` — where environment variables come from in different invocation paths, and what mechanisms are reliable (`/etc/environment`, `BASH_ENV`, `/etc/zsh/zshenv`, etc.).
- {doc}`shell-config` — bash/zsh startup semantics (login vs non-login, interactive vs non-interactive) and practical configuration patterns that work across tools.
